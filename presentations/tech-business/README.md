# 科技商业演示模板 / Tech-Business Presentation Template

## 概述 / Overview

这是一个专为科技和商业演示设计的 PowerPoint 模板，采用现代化的扁平设计风格，配色专业、布局清晰。

This is a PowerPoint template designed specifically for technology and business presentations, featuring a modern flat design style with professional colors and clear layouts.

## 特性 / Features

### 设计风格 / Design Style
- **16:9 宽屏比例** / 16:9 widescreen aspect ratio
- **扁平化设计** / Flat design
- **专业配色方案** / Professional color scheme
- **充足留白** / Ample whitespace for breathing room

### 颜色方案 / Color Scheme
- **主色 / Primary**: #0B3B71 (深科技蓝 / Deep Tech Blue)
- **辅色 / Secondary**: #0CAAAA (青绿色 / Cyan-Green)
- **背景 / Background**: #F5F7FA (浅灰色 / Light Gray)
- **文本 / Text**: #333333 (深灰色 / Dark Gray)

### 字体建议 / Recommended Fonts
- **中文 / Chinese**: Noto Sans CJK SC, PingFang SC, 或 Microsoft YaHei
- **英文 / English**: Montserrat, Arial, 或 Segoe UI
- **标题字号 / Title Size**: 40-60pt (加粗 / Bold)
- **正文字号 / Body Size**: 22-28pt
- **列表字号 / List Size**: 20-24pt

### 页面布局 / Slide Layouts
模板包含以下幻灯片类型：
1. **标题页** - 封面与标题展示
2. **目录页** - 带编号的议程列表
3. **内容页** - 单列要点列表
4. **双栏页** - 两栏对比内容
5. **技术架构页** - 技术方案展示
6. **结论页** - 深色背景的总结页

Template includes the following slide types:
1. **Title Slide** - Cover and title display
2. **Agenda Slide** - Numbered agenda list
3. **Content Slide** - Single-column bullet points
4. **Two-Column Slide** - Two-column comparison content
5. **Technical Architecture Slide** - Technical solution display
6. **Conclusion Slide** - Summary slide with dark background

## 使用方法 / How to Use

### 方法 1：直接编辑 / Method 1: Direct Editing
1. 打开 `tech-business-template.pptx`
2. 编辑每页内容，替换示例文字
3. 保持现有格式和配色方案
4. 根据需要添加或删除幻灯片

1. Open `tech-business-template.pptx`
2. Edit each slide content, replace sample text
3. Maintain existing format and color scheme
4. Add or remove slides as needed

### 方法 2：作为模板应用 / Method 2: Apply as Template
1. 创建新演示文稿
2. 导入模板的母版设计
3. 应用到新幻灯片

1. Create a new presentation
2. Import template master design
3. Apply to new slides

### 内容编辑建议 / Content Editing Tips
- 保持标题简洁明了（不超过 2 行）
- 每页要点不超过 5 条
- 适当使用图标和图表增强视觉效果
- 保持页面间距和对齐一致

- Keep titles concise (no more than 2 lines)
- No more than 5 bullet points per slide
- Use icons and charts appropriately for visual enhancement
- Maintain consistent spacing and alignment

## 版面规格 / Layout Specifications

- **幻灯片尺寸 / Slide Size**: 10 × 5.625 英寸 (25.4 × 14.3 cm)
- **左右边距 / Left/Right Margins**: 2.54 cm (1 inch)
- **上下边距 / Top/Bottom Margins**: 1.9 cm (0.748 inch)
- **标题区高度 / Title Area Height**: 约 2-2.5 cm
- **内容区间距 / Content Area Spacing**: 0.5-1 cm
