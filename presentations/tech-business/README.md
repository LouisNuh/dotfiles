# Tech-Business PPT Template

## 概述 / Overview

这是一个专业的科技商业风格 PowerPoint 演示模板，采用现代扁平化设计，适用于技术产品发布、商业提案、项目汇报等场景。

This is a professional tech-business style PowerPoint presentation template with modern flat design, suitable for technical product launches, business proposals, project reports, and more.

## 设计规范 / Design Specifications

### 配色方案 / Color Scheme

- **主色 Primary**: `#0B3B71` (深科技蓝 Deep Tech Blue)
- **辅色 Secondary**: `#0CAAAA` (青绿色 Cyan-Green)
- **背景色 Background**: `#F5F7FA` (中性灰 Neutral Gray)
- **文字色 Text**: `#3C3C3C` (深灰色 Dark Gray)
- **强调色 Accent**: 白色 White

### 字体规范 / Font Guidelines

**推荐字体 Recommended Fonts:**
- **中文 Chinese**: Noto Sans CJK SC / PingFang SC / Microsoft YaHei
- **英文 English**: Montserrat / Arial / Helvetica

**字体大小 Font Sizes:**
- 标题 Title: 56-64pt (加粗 Bold)
- 副标题 Subtitle: 28-36pt
- 正文 Body: 22-28pt
- 项目符号 Bullet Points: 20-24pt

### 版式规范 / Layout Specifications

- **幻灯片比例 Aspect Ratio**: 16:9
- **页边距 Margins**: 
  - 左右 Left/Right: 2.54cm
  - 上下 Top/Bottom: 1.9cm
- **对齐方式 Alignment**: 网格对齐，统一间距

## 模板内容 / Template Contents

本模板包含以下幻灯片类型：

This template includes the following slide types:

1. **标题页 Title Slide** - 封面页，展示演示主题
2. **内容页 Content Slide** - 项目要点列表，使用圆形符号
3. **两栏对比页 Two-Column Layout** - 对比展示不同方案或观点
4. **数据展示页 Data/Metrics Slide** - 关键指标可视化展示
5. **结论页 Conclusion Slide** - 总结与致谢页面

## 使用方法 / How to Use

### 方法一：直接编辑 / Method 1: Direct Editing

1. 打开 `tech-business-template.pptx` 文件
2. 直接替换示例文本为您的内容
3. 保持现有的颜色和字体风格
4. 根据需要添加或删除幻灯片

### 方法二：另存为模板 / Method 2: Save as Template

1. 打开 `tech-business-template.pptx`
2. 选择 "文件" > "另存为"
3. 选择保存类型为 "PowerPoint 模板 (.potx)"
4. 保存后可在新建演示时选择该模板

### 最佳实践 / Best Practices

- **字体安装**: 为了获得最佳显示效果，建议在系统中安装 Montserrat 和 Noto Sans CJK SC 字体
- **图标使用**: 使用扁平化图标替代手绘风格，保持视觉一致性
- **留白空间**: 保持充足的留白，避免内容过于拥挤
- **配色一致**: 所有新增元素都应使用指定的配色方案
- **文本对齐**: 使用网格对齐确保元素位置整齐

## 注意事项 / Important Notes

### 字体许可 / Font Licensing

- 本模板使用的字体应遵守相应的许可协议
- Noto Sans 字体采用 SIL Open Font License
- 商业使用前请确认字体授权

- 所有文本框均可编辑
